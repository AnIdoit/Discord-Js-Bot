# Discord-Js-Bot
The Discord bot I'm writing in JavaScript(Discord.js)

- dependencies
  - "discord.js": "^11.1.0"

### Commands
- owner(isn't listed in the help command)
  - guildleave
  - say

- moderators
  - ban
  - kick
  - mute

- others
  - useful
    - help
    - guildinfo
  - less useful
    - ping
    - random [number(1-10)/letter/phrase]
